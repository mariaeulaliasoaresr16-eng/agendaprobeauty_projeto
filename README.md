# AgendaPro Beauty

API desenvolvida para o gerenciamento de agendamentos de serviços de beleza.

## Tecnologias Utilizadas
- Node.js
- Express.js
- MySQL (Script DDL incluso)
- Git e GitHub

## Estrutura da Sprint 1
- Configuração inicial do servidor Express.
- Modelagem e criação das tabelas relacionais do banco de dados.
- Organização das pastas do projeto seguindo o padrão MVC/Camadas.

## Notas de Desenvolvimento
Este projeto foi desenvolvido com o suporte de ferramentas de Inteligência Artificial (ChatGPT e Gemini) como ambiente de co-piloto e monitoria para a estruturação inicial da API, lógica de controle de versão (Git) e validação do banco de dados relacional.
